The hard way (any client)
2. Locate your Teeworlds configuration file

Follow instructions on Teeworlds Docs page.

3. Edit your configuration file

{% if not user.is_authenticated %} After logging into Teerace website, go to settings page.
Locate "API token" tab and copy your token.
{% endif %} Go to your teeworlds configuration file and add at the end of this file:

bind BUTTON "rcon_auth teerace:{{ user.profile.api_token|default:"your_api_key" }}"

Replace BUTTON with the button of your choice.

3. Configure your client

After launching the game, go to Settings tab. Then open Race tab. Fill in your username and password and press Retrieve API token.
