Some good news

(posted by: chmielu) | 2008-10-06

Well, my internet is working again so I could commit my change. These changes includes loads of smaller changes here and there but the biggest change is this.

Client and server side demo recording! Yes, you read it right. This includes a nice demo viewer with seekbar (scrubbable), pause, play, slow motion and fast forward options.

There is a few quirks with it and the file size hasn't been optimized yet but it will definitely make be in the 0.5.0 release. An hour or two extra with it will make it really awesome.

Until then, cheerio.
